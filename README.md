## ✨ Funcionalidades

- Design responsivo com breakpoints para mobile, tablet e desktop
- Menu hambúrguer para navegação em telas pequenas
- Efeito aurora animado com CSS `@keyframes`
- Tipografia fluida com `clamp()` — sem saltos em nenhuma resolução
- Scroll suave entre seções com destaque automático do link ativo
- Formulário de contato integrado via Formspree
- Acessibilidade: atributos ARIA, navegação por teclado e `prefers-reduced-motion`
